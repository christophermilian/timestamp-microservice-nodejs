Timestamp Microservice

Available Endpoints:

  • http://localhost:49580/api/:date?

Example Usages and Outputs:

  • http://localhost:49580/api/2023-10-06
    • { "unix": 1696550400000, "utc": "Fri, 06 Oct 2023 00:00:00 GMT" }
  • http://localhost:49580/api/1703462400000
    • { "unix":1703462400000, "utc": "Mon, 25 Dec 2023 00:00:00 GMT" }
  • http://localhost:49580/api/
    • { "unix": 1701039296038, "utc": "Sun, 26 Nov 2023 22:54:56 GMT" }
